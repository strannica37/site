Исследование функции с помощью производной

Производная позволяет определить поведение функции: где она возрастает и убывает, где достигает максимумов и минимумов, как выглядит её график. Это один из важнейших разделов для ЕГЭ по математике.

Критические точки

Критическими точками функции f(x) называются точки, в которых производная f'(x) равна нулю или не существует (при условии, что функция в этих точках определена).

Критические точки: f'(x) = 0 или f'(x) не существует

Критические точки — это «подозрительные» точки, в которых функция может иметь экстремум.

Промежутки возрастания и убывания

f'(x) > 0 на промежутке ⇒ f(x) возрастает на этом промежутке
f'(x) < 0 на промежутке ⇒ f(x) убывает на этом промежутке

Алгоритм нахождения промежутков монотонности:

  • Найти область определения функции.
  • Вычислить производную f'(x).
  • Найти критические точки (f'(x) = 0 и точки, где f'(x) не существует).
  • Расставить критические точки на числовой прямой и определить знак f'(x) на каждом промежутке.

Экстремумы функции (первая производная)

Достаточное условие экстремума (метод первой производной):

Если f'(x) меняет знак с «+» на «−» при переходе через точку x₀, то x₀ — точка максимума.

Если f'(x) меняет знак с «−» на «+» при переходе через точку x₀, то x₀ — точка минимума.

Если знак производной не меняется при переходе через критическую точку, экстремума в ней нет.

Вторая производная и выпуклость

Вторая производная f''(x) характеризует выпуклость графика:

f''(x) > 0 ⇒ график выпуклый вниз (вогнутый) — «чаша»
f''(x) < 0 ⇒ график выпуклый вверх — «горка»

Точка перегиба — это точка, в которой f''(x) меняет знак (выпуклость графика меняется).

Достаточное условие экстремума (вторая производная): если f'(x₀) = 0 и f''(x₀) ≠ 0, то:

  • f''(x₀) > 0 ⇒ x₀ — точка минимума;
  • f''(x₀) < 0 ⇒ x₀ — точка максимума.

Полная схема исследования функции

  • 1. Найти область определения.
  • 2. Найти производную f'(x).
  • 3. Найти критические точки.
  • 4. Определить знак f'(x) на промежутках между критическими точками.
  • 5. Определить промежутки возрастания/убывания и точки экстремума.
  • 6. Вычислить значения функции в точках экстремума.
  • 7. При необходимости: найти f''(x), промежутки выпуклости и точки перегиба.

Примеры

Пример 1. Найти промежутки возрастания и убывания функции f(x) = x³ − 3x + 2.

Решение.
f'(x) = 3x² − 3 = 3(x² − 1) = 3(x − 1)(x + 1).
f'(x) = 0 при x = −1 и x = 1.

Определим знаки f'(x):
— при x < −1: f'(x) = 3(−)(−) = (+) > 0 — возрастание;
— при −1 < x < 1: f'(x) = 3(−)(+) = (−) < 0 — убывание;
— при x > 1: f'(x) = 3(+)(+) = (+) > 0 — возрастание.

Ответ: возрастает на (−∞; −1) и (1; +∞), убывает на (−1; 1).
Пример 2. Найти точки экстремума функции f(x) = x⁴ − 8x² + 3.

Решение.
f'(x) = 4x³ − 16x = 4x(x² − 4) = 4x(x − 2)(x + 2).
f'(x) = 0 при x = −2, x = 0, x = 2.

Знаки f'(x) на промежутках:
— x < −2: f'(x) = 4(−)(−)(−) < 0 — убывание;
— −2 < x < 0: f'(x) = 4(−)(−)(+) > 0 — возрастание;
— 0 < x < 2: f'(x) = 4(+)(−)(+) < 0 — убывание;
— x > 2: f'(x) = 4(+)(+)(+) > 0 — возрастание.

x = −2: смена с «−» на «+» → точка минимума, f(−2) = 16 − 32 + 3 = −13.
x = 0: смена с «+» на «−» → точка максимума, f(0) = 3.
x = 2: смена с «−» на «+» → точка минимума, f(2) = 16 − 32 + 3 = −13.

Ответ: максимум в точке (0; 3), минимумы в точках (−2; −13) и (2; −13).
Пример 3. Найти количество точек экстремума функции f(x) = x·e−x.

Решение.
f'(x) = e−x + x·(−e−x) = e−x(1 − x).
Так как e−x > 0 всегда, знак f'(x) определяется множителем (1 − x):
— при x < 1: f'(x) > 0 — возрастание;
— при x > 1: f'(x) < 0 — убывание.

В точке x = 1 производная меняет знак с «+» на «−», значит, x = 1 — точка максимума.

Ответ: одна точка экстремума (максимум при x = 1).
Пример 4. На рисунке изображён график производной функции f(x), определённой на отрезке [−4; 6]. Найти количество точек максимума функции f(x) на этом отрезке.

Решение (рассуждение).
Точки максимума — это точки, где производная меняет знак с «+» на «−» (т.е. график f'(x) переходит из положительной области в отрицательную, пересекая ось Ox сверху вниз).
Нужно посчитать количество таких переходов на графике f'(x).

Метод: находим точки, где график f'(x) пересекает ось Ox сверху вниз. Каждая такая точка даёт точку максимума исходной функции.
