Геометрический смысл производной

Геометрический смысл производной состоит в том, что она равна угловому коэффициенту (тангенсу угла наклона) касательной к графику функции в данной точке. Это одно из ключевых понятий для задач ЕГЭ.

Касательная к графику функции

Касательная к графику функции y = f(x) в точке x = a — это прямая, которая «наилучшим образом» приближает график вблизи этой точки.

f'(a) = tg α, где α — угол наклона касательной к оси Ox

Значение f'(a) называют угловым коэффициентом касательной (k = f'(a)).

  • Если f'(a) > 0, касательная наклонена вправо-вверх (функция возрастает).
  • Если f'(a) < 0, касательная наклонена вправо-вниз (функция убывает).
  • Если f'(a) = 0, касательная горизонтальна.

Уравнение касательной

Уравнение касательной к графику функции y = f(x) в точке x = a:

y = f(a) + f'(a) · (x − a)

Для составления уравнения касательной нужно знать:

  • Точку касания: x = a, y = f(a).
  • Угловой коэффициент: k = f'(a).

Нормаль к графику

Нормаль — это прямая, перпендикулярная касательной в точке касания. Если угловой коэффициент касательной равен k = f'(a) ≠ 0, то угловой коэффициент нормали:

kн = −1 / f'(a)

Уравнение нормали:

y = f(a) − (1/f'(a)) · (x − a)

Чтение производной по графику

На ЕГЭ часто встречаются задачи, где по графику функции нужно определить знак или значение производной:

  • Если график идёт вверх (слева направо), производная положительна.
  • Если график идёт вниз, производная отрицательна.
  • В точках максимума и минимума производная равна нулю.
  • Чем круче график, тем больше абсолютное значение производной.

Примеры

Пример 1. Написать уравнение касательной к графику функции f(x) = x³ − 2x + 1 в точке x = 1.

Решение.
1) Найдём значение функции: f(1) = 1 − 2 + 1 = 0.
2) Найдём производную: f'(x) = 3x² − 2.
3) Угловой коэффициент: f'(1) = 3 − 2 = 1.
4) Уравнение касательной: y = 0 + 1·(x − 1) = x − 1.

Ответ: y = x − 1.
Пример 2. Написать уравнение касательной к графику f(x) = ex в точке x = 0.

Решение.
1) f(0) = e⁰ = 1.
2) f'(x) = ex, поэтому f'(0) = 1.
3) Касательная: y = 1 + 1·(x − 0) = x + 1.

Ответ: y = x + 1.
Пример 3. К графику функции f(x) = x² провели касательную с угловым коэффициентом k = 6. Найти точку касания.

Решение.
f'(x) = 2x. Нам нужно f'(a) = 6, то есть 2a = 6, откуда a = 3.
f(3) = 9. Точка касания: (3; 9).

Ответ: (3; 9).
Пример 4. Найти уравнение касательной к графику функции f(x) = √x в точке x = 4.

Решение.
1) f(4) = √4 = 2.
2) f'(x) = 1/(2√x), поэтому f'(4) = 1/(2·2) = 1/4.
3) Касательная: y = 2 + (1/4)(x − 4) = 2 + x/4 − 1 = x/4 + 1.

Ответ: y = x/4 + 1.
Пример 5. Написать уравнение нормали к графику функции f(x) = x² + 3x в точке x = 1.

Решение.
1) f(1) = 1 + 3 = 4.
2) f'(x) = 2x + 3, f'(1) = 5.
3) Угловой коэффициент нормали: kн = −1/5.
4) Уравнение нормали: y = 4 − (1/5)(x − 1) = 4 − x/5 + 1/5 = −x/5 + 21/5.

Ответ: y = −x/5 + 21/5.
Пример 6. На графике функции y = f(x) отмечена точка, в которой касательная имеет угловой коэффициент −2. Укажите значение f'(x₀).

Решение.
По определению геометрического смысла производной, угловой коэффициент касательной в точке x₀ равен f'(x₀). Следовательно, f'(x₀) = −2.

Ответ: f'(x₀) = −2.
