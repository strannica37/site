Производная сложной функции

Большинство функций, встречающихся в задачах ЕГЭ, являются сложными (композицией двух или более функций). Для их дифференцирования используется правило цепочки.

Что такое сложная функция

Функция y = f(g(x)) называется сложной функцией (или композицией функций). Здесь g(x) — внутренняя функция, а f(u) — внешняя функция (где u = g(x)).

Например, y = sin(3x + 1): внешняя функция — sin(u), внутренняя — u = 3x + 1.

Правило дифференцирования сложной функции (цепное правило)

(f(g(x)))' = f'(g(x)) · g'(x)

Словами: производная сложной функции равна произведению производной внешней функции (вычисленной от внутренней) на производную внутренней функции.

Для цепочки из трёх функций y = f(g(h(x))):

y' = f'(g(h(x))) · g'(h(x)) · h'(x)

Часто встречающиеся формулы

Применение правила цепочки к стандартным внешним функциям (u = u(x) — внутренняя функция):

  • (un)' = n · un−1 · u'
  • (sin u)' = cos u · u'
  • (cos u)' = −sin u · u'
  • (tg u)' = u' / cos²u
  • (eu)' = eu · u'
  • (au)' = au · ln a · u'
  • (ln u)' = u' / u
  • (√u)' = u' / (2√u)

Примеры

Пример 1. Найти производную f(x) = (5x − 3)⁷.

Решение. Внешняя функция: u⁷, внутренняя: u = 5x − 3.
f'(x) = 7·(5x − 3)⁶ · (5x − 3)' = 7·(5x − 3)⁶ · 5 = 35·(5x − 3)⁶.

Ответ: f'(x) = 35(5x − 3)⁶.
Пример 2. Найти производную f(x) = sin(4x + π/3).

Решение. Внешняя: sin(u), внутренняя: u = 4x + π/3.
f'(x) = cos(4x + π/3) · 4 = 4cos(4x + π/3).

Ответ: f'(x) = 4cos(4x + π/3).
Пример 3. Найти производную f(x) = e−x².

Решение. Внешняя: eu, внутренняя: u = −x².
f'(x) = e−x² · (−x²)' = e−x² · (−2x) = −2x·e−x².

Ответ: f'(x) = −2x·e−x².
Пример 4. Найти производную f(x) = ln(x² + 1).

Решение. Внешняя: ln(u), внутренняя: u = x² + 1.
f'(x) = (x² + 1)' / (x² + 1) = 2x / (x² + 1).

Ответ: f'(x) = 2x / (x² + 1).
Пример 5. Найти производную f(x) = cos³(2x).

Решение. Здесь тройная композиция: f(x) = u³, где u = cos(v), v = 2x.
f'(x) = 3cos²(2x) · (cos(2x))' = 3cos²(2x) · (−sin(2x)) · 2 = −6cos²(2x)·sin(2x).

Ответ: f'(x) = −6cos²(2x)·sin(2x).
Пример 6. Найти производную f(x) = √(1 − 4x²).

Решение. Внешняя: √u, внутренняя: u = 1 − 4x².
f'(x) = (1 − 4x²)' / (2√(1 − 4x²)) = (−8x) / (2√(1 − 4x²)) = −4x / √(1 − 4x²).

Ответ: f'(x) = −4x / √(1 − 4x²).
Пример 7. Найти производную f(x) = 3sin x.

Решение. Внешняя: 3u, внутренняя: u = sin x.
f'(x) = 3sin x · ln 3 · cos x.

Ответ: f'(x) = 3sin x · ln 3 · cos x.
Пример 8. Найти производную f(x) = ln(sin x).

Решение. Внешняя: ln(u), внутренняя: u = sin x.
f'(x) = (sin x)' / sin x = cos x / sin x = ctg x.

Ответ: f'(x) = ctg x (при sin x > 0).
